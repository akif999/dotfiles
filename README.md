# New Computer startup instructions

is following

* (1) Downloading dotfiles from GitHub(This repository).
* (2) Execute `winget import winget.json`.
* (3) Login each applications.
  * Chrome, Discord
* (4) Reboot your computer.
* (5) Installing applications without winget.
  * Google Drive, YouTube Music
  * caps2ctrl, cltc, Everything
  * LINE
* (6) Configuring essential settings.
  * Display, Mouse, Default Application
  * inserting space
* (7) Downloading OneDrive files
* (8) Restore non-synchronized files
  * user/work
* (9) Installing Fonts
  * https://itouhiro.github.io/mixfont-mplus-ipa/migu/
* (10) Add startup task to TaskScheduler
  * AHKStartUp
* (11) WSL export/import
* (12) Restore Windows Terminal configuration
* (13) Restore PowerShell profile
* (14) Restore Environment variable

* appendix
  * Vim settings
    * Set .vimrcs to user directory
    * Installing vim-plug manually
      * https://github.com/junegunn/vim-plug?tab=readme-ov-file
    * Installing colorscheme manually
      * https://github.com/rafi/awesome-vim-colorschemes
